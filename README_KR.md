# LangChain 강좌 - 한국어 가이드

## 🎓 프로젝트 소개

LangChain과 LangGraph를 활용한 실전 AI 에이전트 개발 과정입니다.
실제 API와 실용적인 기술을 사용하여 7개의 완성도 높은 프로젝트를 구축합니다.

## ✅ 환경 구축 완료

현재 개발 환경이 모두 설정되었습니다:
- ✅ GitHub 저장소 클론
- ✅ Python 3.13.3
- ✅ uv 패키지 매니저
- ✅ 프로젝트 의존성 설치
- ✅ Git 설정

## 🚀 빠른 시작

### 1. API 키 설정
\`.env\` 파일을 열고 실제 API 키를 입력하세요:
\`\`\`bash
OPENAI_API_KEY=your_actual_api_key
\`\`\`

### 2. 프로젝트 실행
\`\`\`bash
uv run python main.py
\`\`\`
